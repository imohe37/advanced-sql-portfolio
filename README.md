# Advanced SQL Database Engineering Portfolio
**Author:** Imoh Ezekiel (Lord Zico)  
**Level:** Associate Database Developer / Data Engineer  

Welcome to my SQL optimization and database architecture portfolio. This repository contains end-to-end database solutions, schema designs, and performance tuning implementations across three distinct business domains. Each project contains production-ready scripts alongside full structural documentation.

---

## 🛠️ Core Technical Competencies Demonstrated
* **DDL & Relational Schema Design:** Creating relational tables from scratch with strict data integrity (`PRIMARY KEY`, `IDENTITY` sequences, `FOREIGN KEY REFERENCES`).
* **Advanced Analytical Queries:** Structuring complex logic using Common Table Expressions (CTEs), Virtual Views, and Nested Comparative Subqueries.
* **Defensive Database Programming:** Designing robust Stored Procedures featuring conditional argument fallbacks (`OR @Variable IS NULL`) to ensure application layer stability.
* **Performance Tuning & Optimization:** Eliminating heavy table scans by indexing highly volatile transactional datasets with multi-column Composite Non-Clustered Indexes.
